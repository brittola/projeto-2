# Projeto Docker - Práticas de Containerização

Este projeto contém 5 desafios práticos de Docker. Cada um explora conceitos fundamentais de containerização.

Cada diretório contém um README detalhando o que foi feito em cada desafio, os fluxos, arquitetura, entre outros.

## Tecnologias

- Docker
- Docker Compose
- Node.js (Express)
- PostgreSQL
- Redis

## Decisões Técnicas

As tecnologias Node, Postgres e Redis foram escolhidas por familiaridade e simplicidade no uso e implantação
